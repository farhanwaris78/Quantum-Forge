# Installing QuantumForge on Ubuntu

The maintained Ubuntu 20.04-and-later tutorial is:

**[`docs/INSTALLATION.md`](docs/INSTALLATION.md#4-ubuntu-2004-and-later)**

It covers checksum verification, portable no-root installation, `.deb` installation, the `quantumforge` command, safe update/rollback/uninstall, JavaFX/GTK troubleshooting, MobaXterm X11 forwarding, and preservation of `~/.quantumforge` research data.

For Quantum ESPRESSO, thermo_pw, phonopy, BoltzTraP2, XCrySDen, VASP/CASTEP status, LAMMPS and ML-potential environments, read [`docs/SCIENTIFIC_SOFTWARE_GUIDE.md`](docs/SCIENTIFIC_SOFTWARE_GUIDE.md).

Do not use the historical instruction to run a nonexistent `quantumforge.sh` or assume QE executables are bundled. Release 2.0.0 does not bundle external calculation engines.
